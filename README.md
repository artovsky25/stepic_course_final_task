# Final Project: Selenium Page Object Model

### Description
This project implements the Page Object Model (POM) pattern to automate testing for the Selenium1Py e-commerce website. The tests verify product addition to the basket, success messages, and price consistency.

### Requirements
* Python 3.10+
* Pytest
* Selenium
* Chrome Browser & WebDriver
